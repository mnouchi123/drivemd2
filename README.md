# DriveMD2 Frontend

GitHub Pages 用の静的フロントエンドです。

## Current Features

- `Obsidian_Vault` 配下のフォルダ一覧表示
- Markdown ファイルのプレビュー表示
- Markdown ファイルの編集と保存
- 現在フォルダ配下への新規 `.md` 作成
- ブックマーク
- テーマ切替
- フォントサイズ調整
- 目次表示
- プレビュー画面での `ALL` ボタンによる全文選択

## Default Backend

- `https://drivemd2-proxy-6lgmhmjk7a-an.a.run.app`

## Override

設定画面から `API Base URL` を変更できます。  
または `?apiBaseUrl=https://...` のクエリでも上書きできます。
